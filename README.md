### Setup Instructions

1. Create a virtual environment:

    ```bash
    python -m venv .venv
    ```

2. Activate the virtual environment:

    - On macOS/Linux:
      ```bash
      source .venv/bin/activate
      ```
    - On Windows:
      ```cmd
      .venv\Scripts\activate
      ```

3. Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

4. Run the program:

    ```bash
    python main.py
    ```
